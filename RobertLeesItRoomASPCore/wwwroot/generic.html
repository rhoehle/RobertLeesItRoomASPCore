Front-End Development

As the name implies, the "Front-End" is the part of the code that is on the front of the application. It is usually visible to the user in the form of an interface for the user to interact. This is the main purpose of the front as well as present data for the user in a well-managed form.

HTML, CSS, and JavaScript

Everything that you see on a screen is a mix of HTML, CSS, and JavaScript. Those are the 3 main languages that bring you the web.

The front-end developer is the bridge connecting the designer and the back-end developer. The developer's work is to take the final designs of an application from the designer, and code it into the front-end code so the backend developer can understand it and build functionality of that app around it.
